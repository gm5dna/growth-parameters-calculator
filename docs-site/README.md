# Growth Parameters Calculator - User Documentation

This directory contains the user-facing documentation website for the Growth Parameters Calculator, deployed via GitHub Pages.

## Deployment

The documentation is automatically deployed to GitHub Pages from this directory.

**Live URL:** https://gm5dna.github.io/growth-parameters-calculator/

## Local Development

To view the documentation locally, simply open `index.html` in a web browser.

## Structure

- `index.html` - Main documentation page
- `style.css` - Documentation styling
- `README.md` - This file

## Updating Documentation

1. Edit `index.html` to update content
2. Commit changes to the repository
3. Push to GitHub - documentation updates automatically

## GitHub Pages Configuration

To enable GitHub Pages for this repository:

1. Go to repository Settings
2. Navigate to Pages
3. Set Source to "Deploy from a branch"
4. Select branch: `main`
5. Select folder: `/docs-site`
6. Save

The documentation will be available at: https://gm5dna.github.io/growth-parameters-calculator/
